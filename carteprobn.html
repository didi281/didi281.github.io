Probleme de numărare



  • Numim permutare a mulțimii A, orice mulțime ordonată formată cu cele n elemente.

    • Numărul tuturor perutărilor este \( P_n =n!\)

  • Numim aranjament de n elemente luate câte k, orice submulțime ordonată formată cu k din cele n elemente.

    • Numărul tuturor aranjamentelor este \( A_n^k = \frac{n!}{(n-k)!}\)

  • Numim combinare de n elemente luate câte k, orice submulțime formată cu k din cele n elemente.

    • Numărul tuturor combinărilor este \( C_n^k = \frac{n!}{k! \cdot (n-k)!}\)

    Proprietăți ale funcțiilor

  • Formula combinărilor complementare : \(C_n^k=C_n^{n-k}\)

  • Numărul tuturor submulțimilor unei mulțimi cu n elemente este :

    • \(C_n^0+ C_n^1+ C_n^2 + \dots + C_n^n= 2^n\)


    \(C_n^0+ C_n^2+ C_n^4 + \dots = 2^{n-1}\)


    \(C_n^1+ C_n^3+ C_n^5 + \dots = 2^{n-1}\)




    Numărarea funcțiilor


    Fie 2 mulțimi finite cu card A= m și card B=n și funcția f:A \(\rightarrow\) B

  • Numărul tuturor funcțiilor : \({(card B)}^{card A}\)

  • Dacă \(m \le n \) atunci numărul funcțiilor injective este : \(A_n^m\)

  • Dacă \(m=n \) atunci numărul funcțiilor bijective este : \(P_n\)

  • Dacă \(m \le n \) atunci numărul funcțiilor strict crescătoare(descrescătoare) este : \(C_n^m\)

    • Binomul lui Newton

    \( {(a+b)}^n=\sum_{k=0}^n {C_n^k \cdot a^{n-k} \cdot b^k}\)


  • Dezvoltarea B. N. are n+1 termeni.

  • Termenul general al dezvoltării este \(T_{k+1}= C_n^k \cdot a^{n-k} \cdot b^k\)

  • Raportul a 2 termeni consecutivi este \( \frac{T_{k+2}}{T_{k+1}}=\frac{n-k}{k+1} \cdot \frac ba\)
