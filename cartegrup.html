Legi de compoziție



  • Fie (M,*) o structură algebrică.
  • O submulțime \( H \subset M\) se numește parte stabilă a lui M în raport cu „*” dacă \(\forall x,y \in H \rightarrow x*y \in H\)
  • Legea „*” este asociativă dacă \((x * y)*z=x*(y*z), \; \forall x,y,z \in M\)
  • Legea „*” este comutativă dacă \(x*y=y*x, \; \forall x,y \in M\)
  • Legea „*” admite element neutru dacă \( \exists e \in M \; \; x*e=e*x=x, \; \forall x \in M\)
  • Un element x se numește simetrizabil dacă \( \exists x' \in M \; \; x*x'=x'*x=e\)

    • Legi de compoziție

  • (M,*) se numește monoid, dacă legea „*” este asociativă și admite element neutru.
  • Dacă legea „*” este și comutativă, atunci monoidul este comutativ.
  • (G,*) se numește grup, dacă legea „*” este asociativă, admite element neutru și toate elementele sunt simetrizabile.
  • Dacă legea „*” este și comutativă, atunci grupul este comutativ.

    • Morfisme de grupuri


  • Fie \((G1,*) \text{și} (G2, \circ)\) două grupuri având elementele neutre e1 și e2.
  • O funcție \(f: G1 \to G2\) se numește morfism de grupuri dacă \(f(x *y)=f(x) \circ f(y), \; \forall x,y \in G1\)
  • Dacă f este morfism de grupuri și f este bijectivă, atunci f este izomorfism
  • O submulțime \( H \subset M\) se numește subgrup al lui G dacă H este parte stabilă și simetricul oricărui element din H este din H.

    • Inele

  • \( (A,*,\circ)\) se numește inel, dacă \( (A,*)\) este grup abelian cu elementul neutru \(\theta\), \( (A,\circ)\) este monoid cu elementul neutru e, iar legea „\(\circ\)” este distributivă față de „*”.
  • Un inel \((K,*,\circ)\) este corp, dacă \(\forall x \in K, \; x \neq \theta, \; \exists x' \in K \;\) \( x \circ x' = x' \circ x=e\)
  • Fie \((A1, *, \circ)\) și \((A2, \land, \bot)\) cu elementele neutre \(e, \theta\).
  • O funcție \(f : A1 \to A2\) este morfism de inele dacă \(f(x*y)=f(x) \land f(y), \; \forall x,y \in A1\) \(f(x \circ y)=f(x) \bot f(y), \; \forall x,y \in A1\) \(f(e1)=e2\)

    • Inelul claselor de resturi modulo n


  • \(\mathbb{Z}_n=\{\hat 1, \hat 2, \hat 3, \dots \widehat{n-1} \}\)

  • \((\mathbb{Z}_n,+, \cdot)\) este inel comutativ cu
  • elementul nul \(\hat 0\)
  • elementul unitate \(\hat 1\)
  • opusul lui \(\hat a\) este \(\widehat{n-a}\)
  • inversul lui \(\hat a\) este \({\hat a}^{-1}= \hat a \cdot {\hat a}^{-1}={\hat a}^{-1} \cdot \hat a= \hat 1\)
